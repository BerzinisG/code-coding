Kintamieji

Kintamieji skirti saugoti/laikyti ivairią informaciją programos veikimo metu.

  • string - tekstas
  • number - skaičius
  • boolean - loginis
  • array - masyvas
  • object - objektas

Aritmetika

Kažkokie veiksmai tarp kintamūjų

  • +
  • -
  • /
  • **- kėlimas laipsniu
  • **(1/y)- šaknies traukimas
  • % - liekana

Palyginimai

palyginti keletą duomenų, palyginimas gražina true/false

  • >
  • <
  • >=
  • <=
  • == - lygu
  • === - lygu reikšmes ir tipai
  • != - nelygu
  • !== nelygu reikšmes ir tipai
  • ! - (NOT) apverčia boolean reikšmę
  • && - (AND) - tikrina ar abiejuose pusėse salyga lygi
  • || - (or) - ar bent vienoje pusėje salyga atitinka

Funkcijos

Funkcijose esamas kodas vygdosi Tik tuomet, kai f-ja yra iškviečiama.
Ji gali būti iškviečiama dauglį kartų.
F-ja priema "parametrus"(duomenis), su kuriais atlieka kažkokius veiksmus ir tuomet kreipiantis į f-ja yra perduodami "argumentai"(duomenys).
f-ja gali gražinti kažkokį rezultatą.
F-jos gali būti kviečiamos perduotos kaip parametrai ir tuomet iškviečiamos(callback).

  • funcion name(){} - paprasta
  • const name()=> - arraw/lambda
  • function(){} - bevardė f-ja
  • () => {} - bevardė arrow f-ja

Metodai

Tai kažkokio objekto/masyvo f-ja.

Arrows metodai

funkcijos skirtos tik masyvams. yra daug ir ivairių.
masyvai turi iteracinius metodus. Jie skirti sukti f-ja per masyva ir atlikti kažkokius veiksmus su kiekvienu masyvo elementu

Syntax

masyvas.iteracinisMetodas((masyvoElementas, iteracija, visasMasyvas) => {veiksmai})

Metodus įvairovė

  • String
  • Math
  • Number
  • Object

  • Json
  • Global_objects...

Klasės

klasės yra skirtos kurti objektus.
klasei gali perduoti argumentus, kuriuose klasė konstruktoriaus viduje priima ir pasivadina kazkokiais pavadinimais kuriuos naudoja visos klasės viduje

Pagrindiniai klasės principai

  • inkapsuliacja - viskas ko klasei reikia yra jos viduje
  • abstrakcija - supaprastina kuriamo objekto naudojimą
  • paveldėjimas - kuriant klases galima paveldėti kitas klases
  • polimorfizmas - tos pačios klasės objektai gali turėti skirtingus pavadinimus
